# Recipe AI

An AI-powered recipe generator built with React, Node.js, Express, and Google Gemini.

## Features
- Enter ingredients and get AI-generated recipes
- Clean and responsive dark UI

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- AI: Google Gemini 2.5 Flash API

## Getting Started

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev

## Environment Variables
Create a `.env` file in the `server` folder:
GEMINI_API_KEY=your_key_here
